# SuperAI

Локальный trace-first прототип модульной ИИ-системы. Он реализует ограниченные
Ульи, событийный runtime, типизированное хранилище, малый Космос утверждений,
планирование возможностей и безопасное накопление опыта.

## Быстрый запуск

```bash
python3 -m pip install -e '.[dev]'
./dev.sh
```

API доступен по `http://127.0.0.1:8000/docs`, клиент — по `http://localhost:5173`.
Скрипт запускает backend и Vite одновременно; backend можно запустить отдельно
через `python3 -m uvicorn superai.api:app`.

По умолчанию состояние сохраняется в `.superai/`. Его можно переназначить
переменной `SUPERAI_DATA_DIR`.

## Проверка

```bash
python3 -m pytest
cd web && npm install && npm run build
```

## Реализованный вертикальный путь

`POST /api/v1/tasks` создаёт версионированный контракт и durable WorkItem,
выбирает или восстанавливает Улей, извлекает ограниченные claims из Космоса,
строит типизированный план, кодирует ответ, запускает критиков и сохраняет
trace. Диагностический клиент показывает тот же поток в mock/live режимах.

Дополнительно доступны импорт источников (`/api/v1/sources`), Cosmos,
артефакты, freeze/restore Ульев, dead letters, компостирование и
контролируемый lifecycle навыков. Подробные границы и операции находятся в
[`docs/components`](docs/components).
