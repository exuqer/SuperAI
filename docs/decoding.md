# Декодер

`/api/decode` и страница `/decode` собирают фразу из токенов по схеме:

`tokens/text -> роли -> формы слов -> нормальная фраза`

Декодер read-only: он не пишет в checkpoint, не трогает чат и не меняет `/api/analyze`. Но если в checkpoint уже есть обученные связи, он использует их как подсказки для выбора ролей.

## Вход

Приоритет у `tokens`. Если `tokens` пустой, декодер берет токены из `text`.

```json
{
  "text": "компьютер код писать программист",
  "tokens": ["компьютер", "код", "писать", "программист"],
  "lang": "auto",
  "session_id": "optional",
  "turn_id": "optional"
}
```

## Как выбираются роли

- Для русского сначала ищется глагол.
- Субъект выбирается среди существительных и местоимений.
- Остальные токены получают роли `object`, `instrument`, `location`, `complement` или `modifier`.
- Сильнее всего влияют обученные связи из checkpoint, например `CanDo`, `TakesObject`, `UsesInstrument`, `AtLocation`, `HasProperty`.
- Если обученных связей нет, декодер падает обратно на морфологический baseline.

## Примеры

```text
кот, есть, рыба, мясо -> кот ест рыбу и мясо
осень, лист, становиться, желтый -> осенью лист становится жёлтым
компьютер, код, писать, программист -> программист пишет код на компьютере
```

## Обучение для декодера

Отдельную decoder-сеть создавать не нужно. Достаточно обучать существующий checkpoint через `positive_edges` и `layer_targets`.

Рекомендуемый набор связей для русского синтаксиса:

```json
{
  "positive_edges": [
    ["/c/ru/программист", "CanDo", "/c/ru/писать"],
    ["/c/ru/писать", "TakesObject", "/c/ru/код"],
    ["/c/ru/писать", "UsesInstrument", "/c/ru/компьютер"]
  ]
}
```

## Выход

Ответ содержит:

- исходный текст и токены;
- выбранный язык;
- итоговую `sentence`;
- `pattern`;
- список токенов с ролями и поверхностными формами;
- краткий `summary`.

## Ограничения

- Свободный порядок слов поддерживается эвристически и через обученные ролевые связи.
- Несколько субъектов и сложная пунктуация не поддерживаются.
- Английский пока остается простым baseline-режимом.
