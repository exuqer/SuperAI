# ADR 0003: OpenAPI и единые DTO mock/live

Статус: принято.

FastAPI/Pydantic contracts являются источником HTTP-схем и OpenAPI. Vue-клиент
показывает те же DTO как mock fixtures и в live режиме. Неизвестные
совместимые поля допускаются на границе, неизвестная major-версия схемы
отклоняется.

Это позволяет изучать сохранённую трассу без backend и не поддерживать две
расходящиеся модели данных.
