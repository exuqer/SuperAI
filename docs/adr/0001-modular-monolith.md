# ADR 0001: модульный монолит и trace-first границы

Статус: принято.

Первая версия работает одним Python-процессом с FastAPI, SQLite и файловым
object store. Пакеты имеют явные контракты (`contracts`, `runtime`, `storage`,
`hive`, `cosmos`, `execution`, `learning`), но не становятся сервисами без
измеримой причины.

Команды изменяют состояние, запросы читают его, доменные события фиксируют
значимые уже совершившиеся переходы. Все асинхронные границы несут tenant,
task, trace и correlation identifiers. Внешняя доставка at-least-once;
идемпотентность обеспечивается ключом команды и сохранённой очередью.

Причина: это даёт перезапуск, диагностику и тестируемые границы без затрат
распределённой системы до появления профильной необходимости.
