# Scene Components

`SceneComponent` — употребление word-form в конкретной сцене. Он содержит `placement_id`, `token_index`, грамматическую роль, dependency role и confidence.

`StructuralComponent` — неизменяемая часть внутренней структуры cloud. Он не используется для токенов сцены.

POS принадлежит word-form/lexeme. Грамматическая роль принадлежит scene component.
