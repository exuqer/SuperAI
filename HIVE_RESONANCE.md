# Hive resonance

Activation is fast and saturating; stored strength is slow reinforcement. Direct components receive full gain, same-scene context receives a reduced spread, and background components receive only a trace. Mention counts use logarithmic saturation and refractory windows preserve a small accent for rapid repeats.

The MVP computes resonance dynamically for scene/component and component/component pairs using composition overlap, activation and distance. No permanent semantic edges are created. Clusters are optional derived records tied to a reasoning run.
