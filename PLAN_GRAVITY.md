# Минимальный SuperAI: 2D-гравитационное обучение

## Summary

- Полностью заменить текущий продукт минимальным приложением: одна вкладка «Обучение», пустая SQLite и 2D-карта слов.
- Удалить прежние модули, страницы, API, тесты, документацию, benchmark-данные и локальные данные `.superai/`.
- Перенести из ветки `alg2` только регулярное выражение токенизации русских/латинских слов.

## Implementation Changes

- Оставить минимальные `FastAPI + SQLite + Vue/Vite` и создать API:
  - `GET /api/v1/training/space` — текущее пространство слов.
  - `POST /api/v1/training/learn` с `{ "text": "..." }` — обучить пространство и вернуть обновлённые координаты.
  - `DELETE /api/v1/training/space` — очистить слова и вернуть пустую карту.
- SQLite содержит только `words(word PRIMARY KEY, mass, x, y)`; без seed-данных, связей, истории и разметки.
- Токенизировать слова без пунктуации, нормализовать в нижний регистр; обрабатывать предложения отдельно.
- Для каждого уникального слова в одном запросе:
  - новое слово: масса `1.0`;
  - существующее до начала запроса: масса `+0.1` ровно один раз;
  - повтор слова внутри текста не добавляет дополнительную массу.
- Новые слова размещать на средней дистанции вокруг центра известных слов фразы; для полностью новой фразы — вокруг центра карты.
- Постоянно хранить у слова только массу и координаты. Связи между парами не сохранять.
- На каждом обучении все уникальные пары слов каждого предложения получают временный импульс сближения; повтор фразы усиливает сближение через повторный импульс и возросшие массы.
- Выполнять 60 шагов физической симуляции в пространстве `1000×700`:
  - глобальное притяжение: `6000 * m1 * m2 / max(distance, 32)^2`, ниже `0.02` — `0`, максимум `12`;
  - импульс слов одной фразы: `min(7, 0.015 * m1 * m2 * max(distance - 110, 0))`;
  - отталкивание при расстоянии меньше `88`: максимум `20`;
  - демпфирование скорости `0.82`, границы карты не позволяют словам выйти за поле.
- Реализовать карту нативным SVG: планеты-слова, подписи, масса и расходящийся радиальный градиент. Радиус и интенсивность градиента растут с массой, ограничены радиусом `250`, а за границей градиент прозрачен.
- После обучения плавно анимировать переход текущих координат к рассчитанным за `900 мс`; добавить textarea, кнопку «Обучить», статистику слов и кнопку «Сбросить обучение».

## Test Plan

- Проверить токенизацию: `Привет, как дела?` создаёт `привет`, `как`, `дела`.
- Проверить первую тренировку: три слова с массой `1.0` и разными координатами.
- Проверить повторную тренировку: массы становятся `1.1`; повтор слова в одном запросе не даёт двойной прибавки.
- Проверить, что после повторов средняя дистанция между словами фразы уменьшается, но слова не сходятся в одну точку.
- Проверить ограничение сил, градиента и координат конечными значениями.
- Проверить API обучения, пустое начальное состояние и сброс SQLite.
- Проверить Vue-интерфейс: отображение трёх слов, анимированное обновление массы и очистка карты.
- Запустить `pytest`, `npm run test`, `npm run build` и e2e-сценарий обучения/сброса.

## Assumptions

- Единственная вкладка клиента — «Обучение».
- Визуализация строится через SVG; библиотеки визуализации из других веток не требуются, поскольку SVG сохраняет авторитетные координаты алгоритма.
- Пространство и накопленные массы сохраняются между перезапусками до нажатия «Сбросить обучение».
